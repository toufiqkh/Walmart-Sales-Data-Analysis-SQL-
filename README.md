# Walmart Sales Data Analysis (SQL)

## 📌 Project Overview
This project involves a comprehensive exploratory data analysis (EDA) of Walmart sales data using SQL. The goal was to clean raw data and extract meaningful business insights regarding product performance, customer behavior, and regional revenue trends.

## 🛠️ Technical Skills Demonstrated
* **Data Wrangling:** Dropping redundant features and standardizing data types (Float to Int).
* **Advanced Aggregations:** Using `SUM`, `AVG`, and `COUNT` for KPI generation.
* **Window Functions:** Implementing `RANK()` and `PERCENTILE_CONT` for comparative analysis.
* **Subqueries & CTEs:** Calculating complex metrics like "Revenue Percentage per Product Line."

## 📊 Key Business Insights
* **Revenue Drivers:** Identified the percentage contribution of each product line to the total gross income.
* **Customer Segmentation:** Analyzed purchasing patterns by gender, city, and customer type.
* **Efficiency Benchmarking:** Compared city-level performance against the company average to identify high-performing regions.
